A Buluwy Academy é uma plataforma web desenvolvida para centralizar a distribuição de cursos digitais, com foco em escalabilidade, experiência do utilizador e geração de receita.
O sistema permite organizar conteúdos por categorias, disponibilizar vídeos educativos e facilitar o contacto direto com potenciais clientes via WhatsApp, criando um funil simples e eficiente de vendas.
🚀 Principais funcionalidades:
Estrutura de cursos por categorias
Upload e organização de vídeos
Interface responsiva (mobile-first)
Integração com WhatsApp para conversão rápida
Página dinâmica e otimizada para performance
Design moderno com foco em retenção e vendas
🎯 Objetivo do projeto:
Criar uma solução digital escalável para venda de conhecimento, permitindo à Buluwy posicionar-se como referência em formação tecnológica e expandir para múltiplos cursos e mercados.
💡 Visão futura:
Área de membros com login
Sistema de pagamentos integrado
Dashboard de progresso dos alunos
Automação de marketing e leads
Expansão para marketplace de cursos
